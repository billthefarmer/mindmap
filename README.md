# ![MindMap](MindMap.svg) MindMap [![.github/workflows/build.yml](https://github.com/billthefarmer/mindmap/actions/workflows/build.yml/badge.svg)](https://github.com/billthefarmer/mindmap/actions/workflows/build.yml)

Mind Map app

Uses [MindMapView](https://github.com/hegleB/MindMapView) library by [SangJun
Lee](https://github.com/hegleB).

![MindMap](https://github.com/billthefarmer/billthefarmer.github.io/raw/master/images/MindMap.png)

## Intro
Build yourself a map of whatever takes your fancy. The app allows
adding and removing nodes, editing the text and saving and loading
mindmaps.

## Toolbar
Not all items may appear, depending on node selection.

* **Add** &ndash; Add a new node
* **Remove** &ndash; Remove a node
* **Edit** &ndash; Edit the description
* **Fit** &ndash; Fit the display
* **Open file** &ndash; Load a new mindmap
* **Save file** &ndash; Save the mindmap

Most of the entries will pop up a dialog either prompting for a new
description, or checking that you want to remove something.

## Saving and opening files
When you save a file, the chooser will give you an opportunity to
change the file name, which will default to **MindMap**, unless you
have opened a file with a different name. You can also click on an
existing file to overwrite it. If you just click on the **Save**
button, the chooser will create a series of versions &ndash;
**MindMap(1)**, **MindMap(2)** etc. This may vary with different
devices or different versions of Android.

If you open a [JSON](https://www.json.org/json-en.html) file in a file
manager, or share one from another app, MindMap will attempt to open
it and ask if you want to replace an existing one. A dialog will pop
up if the file isn't a MindMap file or if the app can't read it.

## File format
The mindmap files contain text in the
[JSON](https://www.json.org/json-en.html) format, so they can be
edited with care with a text editor. This is the file from the mindmap
above.

